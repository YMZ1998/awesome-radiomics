## Key Insights

1. **pyradiomics** (1407 stars) is the de facto standard for radiomics feature extraction
2. **CBICA/CaPTk** (197 stars) is the most comprehensive cancer imaging analysis platform
3. **Deep learning integration** is a major trend - hybrid models combining radiomics + CNN
4. **IBSI compliance** is increasingly important for standardized research
5. **Multi-modal radiogenomics** is emerging for precision oncology

## Recommended Starting Points

- For feature extraction: **pyradiomics** + **SlicerRadiomics**
- For complete workflow: **CBICA/CaPTk** or **WORC**
- For deep learning: **mAIstro** or **DL_Radiomics_Fusion**
- For standardization: **PySERA** or **MEDimage**
